# Linux Security Auditor (Beta)
A lightweight Python-based security auditing tool designed for Linux environments.

## Overview
This tool is developed by **Azoth-Engineer** to help system administrators perform quick audits on their Linux machines. It focuses on identifying potential vulnerabilities in network configurations and system file permissions.

## Key Features
* **Port Scanner:** Detects active TCP/UDP listening ports.
* **Permission Auditor:** Monitors sensitive system files for insecure settings.
* **Clean Output:** Provides clear logs directly in the terminal.

## How to Run
1. Clone the repository:
   `git clone https://github.com/Azoth-Engineer/linux-security-auditor.git`
2. Run the auditor:
   `python3 auditor.py`
